// Capabilities

System Architecture

End-to-end autonomous sampling and reconstruction pipeline built on ROS2 and GPyTorch.

Gaussian Process Regression

Multiple GP methods including Standard GP, McHutchon NIGP for input noise, and Girard uncertain-input propagation. GPU-accelerated via GPyTorch.

Information-Theoretic Sampling

Adaptive sampling strategies that maximize information gain. Entropy-based path planning for efficient field coverage with positional uncertainty.

PX4 SITL Simulation

Full software-in-the-loop simulation with PX4 autopilot, Gazebo physics, and realistic USV dynamics. Docker containerized for reproducibility.

Real-Time Reconstruction

Live field reconstruction with uncertainty quantification. Web dashboard for monitoring trials, comparing methods, and visualizing results.

Low-Cost Hardware

R/V Karin Valentine: $2,500 ASV platform with GPS, IMU, and water quality sensors. Designed for accessible environmental research.

Containerized Deployment

Docker-based infrastructure with VNC access for each simulation trial. Batch processing support for systematic experimentation.
